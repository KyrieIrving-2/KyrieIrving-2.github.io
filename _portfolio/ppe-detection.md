A real-time PPE (Personal Protective Equipment) detection system for construction site safety monitoring. The system detects 6 categories: person, work_cap, safety_helmet, gloves, work_clothes, and mask.

**Key Features:**
- Real-time video stream processing
- YOLOv8 based detection model
- Deployed on Sophon BM1684/BM1688 edge devices
- ROI compliance alarm with voting debounce
- HTTP/MQTT push notifications
